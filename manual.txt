title - название товара (1. запрещено использование '__' и '___' в названии 2. !уникальное значение! 3. !=NULL)

price - цена товара в рублях (!=NULL)

description - описание товара

path - путь до текстового файла с данными в формате логин:пароль (одна строка используется один раз, после чего удаляется из файла)

type - тип товара:
	type==NULL - ничего не происходит
	type==ru_vds - запускается скрипт получения vds хостинга
	type==string - пользователю отправляется первая строка из текстового файла расположенного по пути в столбце path

------------------------------------

bot_token - токен телеграм-бота
yoo_token - токен кошелька юмани с правами на просмотр истории пополнений
rucapcha_token - токен с сайта rucapcha

time_to_top_up - время на пополнение баланса при создании заявки (в минутах)
max_flows - максимально кол-во потоков при создании аккаунта RuVds
wallet - номер кошелька юмани
support - юзернейм поддержки бота в телеграм (без символа "@")
